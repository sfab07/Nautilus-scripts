# Nautilus-scripts

pdf / audio / video file conversion and processing

Requirements :

Pdftk :
sudo apt install -y pdftk

WebP :
sudo apt-get install -y webp gthumb

eyeD3 :
http://ftp.us.debian.org/debian/pool/main/e/eyed3/eyed3_0.7.10-1_all.deb
